⚠️ Disclaimer

This prediction system is based on an XGBoost machine learning model trained on the SEER database (12,187 esophageal cancer patients, 6.32% lung metastasis rate). Probabilities are calibrated using Platt Scaling on the validation set (selected over Isotonic Regression by lower test Brier score) to better reflect true event likelihood. Risk levels are defined relative to the observed calibrated probability distribution in this cohort: low (<5%), medium (5-15%), and high (≥15%). Even the highest predicted probability in this dataset is modest (around 20%). The model is intended for clinical reference only and has not been externally validated. Prediction results cannot replace professional medical diagnosis and treatment decisions. Please consider the patient's specific clinical situation, imaging findings, and laboratory results for comprehensive evaluation. Model performance may vary across different patient populations. Consult healthcare professionals if you have any questions.
